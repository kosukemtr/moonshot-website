## キックオフミーティング

- 日時: 2026年5月11日(月) 10:00-17:00頃
- 会場: 大阪大学 豊中キャンパス 南部陽一郎ホール
- 形式: 対面開催
- 対象: プロジェクト内部限定

ムーンショット目標6「誤り耐性量子コンピュータのアプリケーション研究開発」プロジェクトのキックオフミーティングを開催予定です。
本イベントは、プロジェクト関係者を対象とした内部限定イベントです。
プロジェクト全体の方向性を共有するとともに、各研究チームのこれまでの取り組みや今後の連携の進め方について議論します。

## 暫定プログラム

| 時間 | 内容 |
|------|------|
| 10:00-11:30 | 各PIによる自己紹介およびこれまでの研究内容の紹介 |
| 11:30-13:30 | ランチ・懇親 |
| 13:30-14:00 | 休憩・自由討論 |
| 14:00-15:00 | FTQC tools の説明 |
| 15:00-15:30 | 休憩・ディスカッション |
| 15:30-16:30 | 大阪大学 ELSI センターより、研究推進における ELSI 的観点についての話題提供 |
| 16:30-17:00頃 | 全体討論・クロージング |

※ プログラムは今後調整の可能性があります。
